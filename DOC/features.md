# Core Features of MovieKart

## 1. User Authentication

* User signup & login
* Secure JWT/Auth.js authentication
* Google OAuth login
* Profile setup with avatar and bio

---

* Unique public username/profile
* Shareable profile link
* User bio and favorite genres
* Display:

* Followers & following system
* User activity timeline
